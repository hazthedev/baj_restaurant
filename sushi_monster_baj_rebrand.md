# Sushi Monster — Rebrand Plan (using B.A.J Restaurant template)

> **Project brief:** Repurpose the existing B.A.J Restaurant MVP (`baj-restaurant.vercel.app`) as a landing site for **Sushi Monster**, a halal Japanese sushi chain (4 KL locations). Same template structure (cult-favorite editorial, signature dish hero, press strip). Different cuisine, halal-confident positioning, multi-location architecture.
>
> **Why this swap works:** B.A.J template was built around (1) one viral signature dish, (2) press features as social proof, (3) cult-favorite editorial mood, (4) confident-quirky brand voice. Sushi Monster has all four — the affordable RM1.90 plate is the viral hook, recent halal certification + TikTok features are the social proof, and the playful "monster" theme fits the editorial confidence. The only mismatch is single-location → multi-location, which we solve by repurposing the B.A.J Story page into a Locations page.
>
> **Pitch positioning:** Multi-location halal Japanese chain. RM4,500 project value (versus single-shop ~RM3,000) because of the locations page complexity and SEO leverage across 4 venues.

## 2. Design Direction (mapping from B.A.J template)

### Color Palette (swap from B.A.J's cheese-yellow)

| Role | B.A.J Original | Sushi Monster Replacement | Hex |
|------|----------------|---------------------------|-----|
| Background (primary) | Bone White | Soft Off-White | `#FAF6F0` |
| Surface | Pure Cream | Pure White | `#FFFFFF` |
| Primary text | Ink Black | Deep Navy | `#1C2742` |
| Secondary text | Soy Brown | Slate | `#5A6478` |
| **Accent (THE color)** | Cheese Yellow `#E8B838` | **Monster Coral** | `#E94B47` |
| Plate Tier 1 | n/a | **Wasabi Green** | `#7AB852` |
| Plate Tier 2 | n/a | **Salmon Red** | `#E94B47` |
| Plate Tier 3 | n/a | **Gold (Special)** | `#D9A441` |
| Highlight | Soft Mustard | Sunshine Yellow | `#F5C84B` |
| Deep contrast | BKT Brown | Soy Black | `#15171F` |

**Why these:**
- **Monster coral** — primary brand accent, ties to salmon nigiri (their hero dish) and the "monster" energy
- **Wasabi green / salmon red / gold** — direct color-coding of the plate tier pricing system (this becomes a brand visual asset, no equivalent in B.A.J)
- **Off-white + navy** — keeps clean, trustworthy mall-family feel (lighter and brighter than B.A.J's bone-white + ink-black)
- **Sunshine yellow** for fun monster moments

**Visual difference from B.A.J:** B.A.J leans warm/earthy/cult-favorite-grungy. Sushi Monster needs to feel **bright, family-friendly, mall-clean** — same template structure, but visually lighter and more playful.

### Typography (mostly keep B.A.J's choices, add one playful display)

- **Display / Headings:** `DM Serif Display` (keep — same editorial energy, works for both brands)
- **Body:** `Inter` (keep)
- **Numerals (prices):** `Inter` bold (keep — affordable casual feel; prices like "RM1.90" are the hero text)
- **Accent / handwritten:** `Caveat` for "from RM1.90" annotations (replaces B.A.J's "since 2018")
- **Optional fun display:** `Bagel Fat One` for super-bold price callouts (e.g., the "RM1.90" hero badge — adds playful monster energy without breaking B.A.J's editorial structure)

### Mood

Halal-Japanese · Family-Friendly · Affordable · Vibrant · Playful · Multi-Location · Trust-Forward

Visual references: Genki Sushi friendly mall branding, colorful tier-priced sushi conveyors, halal-certified family restaurants, playful monster-themed brand identity. NOT: premium omakase austerity, dark Japanese izakaya, sterile chain feel.

## 6. Locations Page Build Spec (most important new component)

This is the page that justifies the higher project value. Build it carefully.

### File structure
```
src/pages/locations.astro
src/components/LocationCard.astro
src/data/locations.json
```

### `locations.json` shape
```json
[
  {
    "id": "setapak-central",
    "name": "Setapak Central",
    "tag": "Flagship",
    "lot": "F12",
    "floor": "1st Floor",
    "address": "67 Jln Taman Ibu Kota, Taman Danau Kota, 53300 Kuala Lumpur",
    "phone": "+60 14-372 2425",
    "whatsapp": "+60 14-372 2425",
    "hours": "10am – 10pm daily",
    "googleMapsEmbedUrl": "...",
    "foodPandaUrl": "https://foodpanda.my/...",
    "image": "/images/locations/setapak-central.jpg"
  },
  // ... 3 more
]
```

### Page structure
- Hero: "Find Your Monster." + subhead
- All-Locations Map (single iframe, all 4 pins) OR 4-image grid if no API key
- Iterate over `locations.json` → render `LocationCard` for each
- Contact form at bottom with location-select dropdown
- Social block

### `LocationCard.astro` structure
- Photo (left col on desktop, full-width on mobile)
- Right col:
  - Tag badge ("Flagship" / "Newest" / etc.)
  - Mall name (large, in DM Serif Display)
  - Lot + floor info
  - Address (italic smaller)
  - Hours
  - Phone
  - Google Maps iframe (small embed, ~300px height)
  - 2 CTAs: "Get Directions" + "Order on FoodPanda"

## Notes for Claude Code

- Read the existing B.A.J site code before starting — understand the components and data structure already in place
- Commit after each phase: `feat(rebrand): swap colors`, `feat(rebrand): tier system`, `feat(rebrand): locations page`
- Default mood for this rebrand: bright, family-friendly, halal-confident, vibrant, multi-location chain (NOT cult-favorite-grungy like B.A.J was)
- The big architectural change is replacing the Story page with a Locations page — don't skip this, it's the entire reason the project value is RM4,500 instead of RM3,000
- Photography is bright and abundant — show stacks of plates, full tables, family meals (NOT B.A.J's blogger-grunge style)
- **Language rule (strict):** All readable content in English. Japanese dish names (sushi, tempura, soba, nigiri, etc.) stay in original spellings. Brand name stays.
- **Halal rule:** Halal certification is a primary trust signal. Surface on home, locations page, and footer. Use cert badge prominently if available.
- **Tier color rule:** The plate tier system (green/red/gold) is part of the brand identity. Use it consistently — always pair color with text label for accessibility. This is your single biggest visual differentiator from B.A.J.
